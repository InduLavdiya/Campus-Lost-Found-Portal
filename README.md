# Campus Lost & Found Portal

A responsive web application that helps students report, search, and recover lost and found items on campus.

---

## 🚀 Live Demo

https://campus-lost-found-portal.web.app

---

## 📂 GitHub Repository

https://github.com/InduLavdiya/Campus-Lost-Found-Portal

---

## ✨ Features

- User Registration & Login
- Forgot Password
- Report Lost Items
- Report Found Items
- Optional Image Upload (Cloudinary)
- View Item Details
- Edit & Delete Own Reports
- Search Reports
- Filter Lost / Found Items
- Responsive Design
- Dark Mode
- Real-time Firestore Database

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase Authentication
- Firebase Firestore
- Firebase Hosting
- Cloudinary
